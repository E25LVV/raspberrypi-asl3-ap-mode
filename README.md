# Raspberry Pi ASL3 AP Mode

Turn Raspberry Pi into a private Wi-Fi AP for portable AllStarLink 3 operation and mobile SSH administration.

---

# เปลี่ยน Raspberry Pi ให้เป็น Wi-Fi AP ส่วนตัว สำหรับ AllStarLink 3

โปรเจคนี้ออกแบบสำหรับเพื่อนๆนักวิทยุสมัครเล่นที่ต้องการใช้งาน AllStarLink 3 แบบ portable
โดยสามารถเชื่อมต่อเข้า Raspberry Pi ผ่าน Wi-Fi ส่วนตัวได้โดยตรง
โดยไม่ต้องพึ่ง router ภายนอกตลอดเวลา

รองรับ:

* Debian 12
* Debian 13
* AllStarLink 3

เหมาะสำหรับ:

* Portable RF Node
* Field Day
* Emergency Communication
* Temporary RF Site
* Mobile SSH Administration
* Raspberry Pi แบบไม่มีจอ (Headless)
